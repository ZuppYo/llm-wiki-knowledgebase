# Philips AUT8940 Installation and First Use

Compiled from [Philips AUT8940 RO Water Filter 101](../Topics/philips-aut8940-ro-water-filter-101.md).

## In the box (typical)

Unit, dual-mode faucet, adapter valve for feed water, power adapter, tubing (white inlet, green MF out, blue RO out, gray drain), drain saddle kit for ~1" pipe, rubber pad for horizontal mount, manuals (EN/TH/VN).

## Plumbing summary

1. **Feed:** Install Philips adapter valve on cold supply; connect **white** inlet to unit (lock collet fittings).
2. **Unit ports (top/rear):** white = in, blue = RO out, green = MF out, gray = drain.
3. **Faucet:** Often replaces existing sink tap (large stem — may need to widen hole or use separate mount). Connect blue/green lines to faucet; selector chooses MF vs RO.
4. **Drain:** Drill saddle on drain pipe; gray line routes RO reject (~30% in demo).

## First-time startup

1. Open feed valve; **wait ~5 min** with power **off** so lines fill.
2. Power on — three indicator lights; brief pump vibration; **auto membrane rinse ~18 s**.
3. **Manual initial flush (first use only):**
   - Select **MF** (droplet icon): run **15 minutes**, close 10 s pause.
   - Select **RO** (glass icon): run **20 minutes**.
4. Normal use afterward: short auto-rinse on each power cycle; no repeat 15/20 min unless manual flush needed.

## Touch panel (AUT8940)

| Action | Control |
|--------|---------|
| MF flush | Short press refresh/droplet button (~1–2 min) |
| FreshPro on | Hold droplet/refresh **5 s** (3 blinks + 2 beeps) |
| FreshPro off | Hold again **5 s** |
| Reset filter life | Hold lower button **3 s** → choose **AQP** (RO) or **4-in-1** (MF) → hold to confirm; cancel: hold both **5 s** |
| Filter status | Steady green = OK; blinking red = nearing end; solid red = replace now |

## Filter replacement

Open front cover; lift QuickTwist cartridge; twist lock on reinstall. Reset corresponding filter counter after change.

## TDS demo (review measurements)

| Source | TDS (approx.) |
|--------|----------------|
| Tap water | ~100–200 |
| MF outlet | ~95–96 |
| RO outlet | ~3–4 |

Taste note in source: remineralized RO not “flat/bitter” like some RO-only units.

## Sizing tip (from source)

400 GPD (~1,514 L/day rated) adequate for normal family drinking; consider AUT8990 if household constantly draws large volumes (e.g. frequent batch drinking water).
