# Philips AUT8990 Features and Specifications

Compiled from [Philips AUT8990 Dual-Mode Water Filter 101](../Topics/philips-aut8990-dual-mode-water-filter-101.md).

## Form factor

- Compact **tankless** under-sink unit — vertical or horizontal mount.
- Requires **mains power** for RO motor (adapter included in kit).

## Panel and sensors

| Feature | Behavior (source) |
|---------|-------------------|
| **Dual filter indicators** | Separate life tracking for **AQP (RO)** and **4-in-1 (MF)** |
| **TDS sensor** | Built-in; auto membrane flush when dissolved solids rise too high |
| **Filter alerts** | Steady = OK; **blinking red** = replace soon; **solid red** = replace now |
| **Reset after change** | Hold panel button to reset counter for replaced cartridge |

## Key specs (this source)

| Item | Value |
|------|--------|
| Model | AUT8990 |
| Rated output | **400 GPD** (~1,500 L/day per source) |
| RO flow | **~2 L/min** (with internal pump) |
| Waste ratio | **~30%** drain / **~70%** product water (efficiency claim) |
| Warranty | **2 years** via Philips Water Official (source recommendation) |

## GPD note vs AUT8940 review

[Philips AUT8940 features](philips-aut8940-features-specs.md) (OHMSHOWstudio) describes AUT8990 as **800 GPD** with similar ~2 L/min RO flow. This T3B review states **400 GPD** for AUT8990. Treat GPD as **source-specific** until confirmed against official Philips specs.

## Use cases highlighted in source

- Non-municipal / groundwater areas needing RO for drinking safety.
- Separating **produce rinse (MF)** from **drinking (RO)** in one install.
- Households wanting display + TDS monitoring without a separate mineral cartridge.
