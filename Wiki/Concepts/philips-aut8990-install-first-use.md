# Philips AUT8990 Installation and First Use

Compiled from [Philips AUT8990 Dual-Mode Water Filter 101](../Topics/philips-aut8990-dual-mode-water-filter-101.md).

## Kit contents (typical)

Unit, dual-output faucet, feed adapter valve, power adapter, tubing (white inlet, green MF, blue RO, gray drain), drain saddle for ~1" pipe, elbows/connectors, manuals. Full install hardware included per source.

## Plumbing summary

1. **Shut off** cold supply under sink.
2. **Adapter valve:** Insert between existing tap stem and original faucet head; white inlet connects to unit (push-to-lock collet — no thread tape required in demo).
3. **Unit ports:** White = inlet; green = MF out; blue = RO out; gray = drain.
4. **Faucet:** Mount on sink (may need hole enlargement) or use separate faucet bracket; connect green/blue lines.
5. **Drain:** Drill saddle on ~1" drain pipe; foam gasket + clamp; insert gray reject line.

Mount unit **vertically or horizontally**; optional 90° elbow to tidy hoses.

## First-time startup sequence

1. Open feed valve; verify connections.
2. Power on — **automatic initial rinse** runs.
3. **Manual first-use flush:**
   - Select **MF** on faucet: run **15 minutes**.
   - Move lever to **center (off)**: wait **10 seconds**.
   - Select **RO**: run **20 minutes**.
4. After sequence, **TDS monitoring** operates normally; high TDS triggers auto flush.

## Ongoing maintenance

- Use **MF** for high-volume rinsing; **RO** for drinking — preserves AQP life (see [dual filter paths](philips-aut8990-dual-filter-paths.md)).
- Replace cartridges when panel shows red; **reset** corresponding filter counter.
- Source recommends purchasing from **Philips Water Official** for 2-year warranty.

## DIY vs installer

Source presents a mock bench demo; real under-sink install is similar. DIY feasible if comfortable with basic plumbing; otherwise hire installer — all parts ship in box.
