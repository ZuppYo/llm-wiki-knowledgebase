# Agent Harness 101

Foundational mikelopster episode on **agent harness**: the infrastructure and rules wrapped around a model so agents act reliably—not model capability alone. Same model in Cursor vs Codex vs Claude Code can behave differently because each product’s harness differs.

**Agenda from source:** harness definition and history (pre-harness → AutoGPT → frameworks); typical harness components; how to build or customize harnesses; Codex and Claude Code as case studies; sets up later [Deep Agent 101](deep-agent-101.md) (harness frameworks on LangChain/LangGraph).

**Prerequisites:** helpful to have seen [LangChain 101](langchain-101.md) / [LangGraph 101](langgraph-101.md) for framework context; not required for the conceptual overview.
