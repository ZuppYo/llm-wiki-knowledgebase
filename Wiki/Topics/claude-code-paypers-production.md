# Claude Code Paypers Production

Talk by Paypers CTO Wasin Watthanasrisong (Creatorsgarten, June 2026): production use of Claude Code from zero on an expense-management startup (~4 months, ~20k businesses). Complements generic harness theory in [Agent Harness 101](agent-harness-101.md) with a **live product** case study.

**Context:** Paypers chose Claude Code over Gemini/Codex at project start; stack stayed familiar (Vercel, Supabase, Terraform) so human review stayed efficient. Product direction still comes from customer conversations—not broad AI prompts (internet signal is noisy).
